A scope is where you can refer to a variable. A block defines a block scope a variable defined inside a block will be defined only inside that block and you can't reference it after the end of block.Before ES6, JavaScript only had function and global scope. This meant that a variable declared inside a function lived inside the function and a variable declared outside a function could live anywhere.

Event loop: An event loop is something that pulls stuff out of the queue and places it onto the function execution stack whenever the function stack becomes empty. The event loop is the secret by which JavaScript gives us an illusion of being multithreaded even though it is single-threaded.
